# MelonSense

MelonSense is a native Android app for judging watermelons with visual cues, knock-test audio, and personal outcome learning.

## Current Scaffold

- Kotlin Android app
- Jetpack Compose UI
- Material 3 theme
- Navigation skeleton
- Placeholder screens:
  - Scan
  - Knock Test
  - Result
  - History
  - Settings

## Build

```bash
./gradlew :app:assembleDebug
```
